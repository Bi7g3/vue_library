# Okul Kütüphane Sistemi (UI)

Bu paket **yalnızca tasarım (UI)** içerir. Veri tabanı, servis, auth entegrasyonu yoktur.

## Kurulum
```bash
npm install
npm run dev
```

## Sayfalar
- `/` Login
- `/app/dashboard` Dashboard (sol menü + kartlar)
- `/app/books` Kitap İşlemleri (şablon)
- `/app/loans` Ödünç İşlemleri (şablon)
- `/app/reports` Raporlar (şablon)
